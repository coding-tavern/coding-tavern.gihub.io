Asi es como empieza este blog, de manera rebuscada como varias cosas de la programacion.…

Y como dice el titulo, “Hola mundo”, frase tan quemada por los tutoriales en internet y canales de youtube.

Y de lo que tambien estamos quemados es de uno de los tipicos problemas en la programacion que ya deberian estar resueltos, es impresionante que hoy en dia los lenguajes no tengan incorporadas herramientas para resolver los problemas mas basicos, en varios lenguajes dar vuelta una cadena es un parto.

Empecemos con la tipica implementacion, que aprendimos en la escuela (en java):

class ReverseString {
    public static void main(String[] args)
    {
        String input = "GeeksforGeeks";
 
        // getBytes() method to convert string
        // into bytes[].
        byte[] strAsByteArray = input.getBytes();
 
        byte[] result = new byte[strAsByteArray.length];
 
        // Store result in reverse order into the
        // result byte[]
        for (int i = 0; i < strAsByteArray.length; i++)
            result[i] = strAsByteArray[strAsByteArray.length - i - 1];
 
        System.out.println(new String(result));
    }
}

Una idea bastante sencilla, tomar la cadena como un array, crear un array nuevo con las posiciones invertidas, y volverlo a convertir en cadena.

La idea termina siendo la misma en todos los lenguajes, usando diferentes implementaciones, en js como en el titulo:

function reverse(s){
    return s.split("").reverse().join("");
}

En c#:

public static string Reverse( string s )
{
    char[] charArray = s.ToCharArray();
    Array.Reverse( charArray );
    return new string( charArray );
}

En python:

>>> 'hello world'[::-1]
'dlrow olleh'

Algunas pueden ser mas elegantes que otras, pero coinciden en un mismo punto, no hay una implementacion nativa del lenguaje. Hoy en dia me parece ridiculo, y aunque entiendo que dar vuelta una cadena no es algo que se hace todos los dias en el trabajo, 쯡o va siendo hora de que lo implementen? sale un nuevo ecmascript casi anualmente, python se actualiza relativamente seguido, 쯇or que es tan dificil agregar el metodo cadenita.reverse() ?
